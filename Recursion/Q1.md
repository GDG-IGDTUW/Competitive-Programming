# 1. Print Numbers Recursively

### Description
Given an integer n, print numbers from 1 to n using recursion.

### Input
- One integer n (1 ≤ n ≤ 1000)

### Output
Print numbers from 1 to n separated by spaces.

### Sample
**Input**
5

**Output**
1 2 3 4 5

### Code (C++)
```cpp
#include <bits/stdc++.h>
using namespace std;

void print(int x, int n) {
    if (x > n) return;
    cout << x << " ";
    print(x + 1, n);
}

int main() {
    int n;
    cin >> n;
    print(1, n);
}

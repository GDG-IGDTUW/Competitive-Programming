## 3. Palindrome Check (Recursive)

### Problem Name
Palindrome Check

### Problem Description
Given a string \(S\), determine whether it is a **palindrome** using recursion.

A palindrome reads the same forward and backward.

### Input Description
- A single string \(S\)

### Output Description
- Print `YES` if the string is a palindrome, otherwise print `NO`

### Sample Test Cases
**Input**
```
level
```
**Output**
```
YES
```

**Input**
```
codeforces
```
**Output**
```
NO
```

### Constraints
- \(1 \le |S| \le 10^5\)
- String contains lowercase English letters

  **code**
  #include <iostream>
  #include<string>
   using namespace std;

bool isPalindrome(string s, int start, int end) {
    // Base case: single character or empty string
    if (start >= end) {
        return true;
        
   // If characters at both ends are not equal
    if (s[start] != s[end]) {
        return false;
    }
    // Call again 
    return isPalindrome(s, start + 1, end - 1);
    }

int main() {
    string s;
    cout << "Enter a string: ";
    cin >> s;
     if (isPalindrome(s, 0, s.length() - 1)) {
        cout << "YES";
    } else 
        cout << "NO";
    return 0;
}

# 6. Equal Difference Window

### Description

You are given an array of n integers.

Find the length of the longest contiguous subarray such that:

The difference between the maximum and minimum element in the subarray is at most k.

### Input

- The first line contains integers n and k (1 ≤ n ≤ 2×10⁵).
- The second line contains n integers a[i] (1 ≤ a[i] ≤ 10⁹).

### Output

Print the maximum possible length.

### Sample

**Input:**
```
6 3
1 3 6 4 2 5
```

**Output:**
```
4
```

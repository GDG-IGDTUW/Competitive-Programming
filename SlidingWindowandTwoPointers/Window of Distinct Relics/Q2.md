# 2. Window of Distinct Relics

### Description

You are given an array of n integers representing relic types.

You want to find the length of the longest contiguous subarray such that:

- The subarray contains at most k distinct values.

### Input

- The first line contains integers n and k (1 ≤ n ≤ 2×10⁵).
- The second line contains n integers a[i] (1 ≤ a[i] ≤ 10⁹).

### Output

Print the maximum possible length.

### Sample

**Input:**
```
7 2
1 2 1 3 4 2 3
```

**Output:**
```
3
```

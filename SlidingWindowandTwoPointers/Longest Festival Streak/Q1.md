# 1. Longest Festival Streak

### Description

You are given an array of n integers representing daily happiness values.
You want to choose a contiguous subarray such that:

- The sum of elements in the subarray is at most k.
- The length of the subarray is maximized.

Find the maximum possible length.

### Input
- The first line contains integers n and k (1 ≤ n ≤ 2×10⁵, 1 ≤ k ≤ 10¹⁸).
- The second line contains n integers a[i] (1 ≤ a[i] ≤ 10⁹).

### Output
Print a single integer — the maximum length.

### Sample
**Input:**
```
5 7
2 1 3 2 1
```
**Output:**
```
3
```

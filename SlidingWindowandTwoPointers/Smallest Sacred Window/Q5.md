# 5. Smallest Sacred Window

### Description

You are given an array of n integers.
Find the smallest contiguous subarray such that:

- The sum of elements is greater than or equal to k.

If no such subarray exists, print -1.

### Input

- The first line contains integers n and k (1 ≤ n ≤ 2×10⁵).
- The second line contains n integers a[i] (1 ≤ a[i] ≤ 10⁹).

### Output

Print a single integer.

### Sample

**Input:**
```
5 15
1 2 3 4 5
```

**Output:**
```
5
```

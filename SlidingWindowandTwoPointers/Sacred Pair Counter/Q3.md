# 3. Sacred Pair Counter

### Description

You are given a sorted array of n integers.
You want to count the number of pairs (i, j) such that:

- i < j
- a[i] + a[j] ≤ x

### Input

- The first line contains integers n and x (1 ≤ n ≤ 2×10⁵).
- The second line contains n integers a[i] (1 ≤ a[i] ≤ 10⁹).

### Output

Print the number of valid pairs.

### Sample

**Input:**
```
5 6
1 2 3 4 5
```

**Output:**
```
6
```
